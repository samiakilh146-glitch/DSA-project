# Heap and Priority Queue Implementation in Java

## 📌 Project Description
This project demonstrates the implementation of Min Heap and Max Heap from scratch in Java without using built-in PriorityQueue. It also includes problems solved using Java PriorityQueue.

---

## 📚 Features

### 1. Min Heap Implementation
- insert()
- extractMin()
- getMin()
- heapifyUp()
- heapifyDown()

### 2. Max Heap Implementation
- Same structure as Min Heap
- Only difference is reversed priority (Max element at root)

### 3. Java PriorityQueue Usage
- Min Priority Queue
- Max Priority Queue using Collections.reverseOrder()

### 4. Problem Solving
- K Largest Elements using Priority Queue
- K Smallest Elements concept
- Hospital Emergency System (Priority-based patient handling)

---

## 🏥 Case Study
Hospital Emergency System using PriorityQueue where patients are treated based on severity and arrival time.
